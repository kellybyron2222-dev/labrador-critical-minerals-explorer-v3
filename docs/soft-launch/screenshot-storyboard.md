# Soft-launch screenshot storyboard

Capture 5–10 stills from the live app (1920×1080 or phone crop). Prefer real Labrador AOIs.

| # | Story | How to capture |
|---|--------|----------------|
| 1 | Hero overview | Positron basemap, Labrador full extent, Critical Minerals on |
| 2 | Claims + MODS | Zoom to a district; claims + MODS commodity colors visible |
| 3 | Hard exclusions | Hard exclusions on; parks / water supplies readable |
| 4 | Screen report | Run **Screen this area** → screenshot the in-app report panel |
| 5 | Radius AOI | Screen → **Radius** 25 km circle + pin |
| 6 | Infrastructure | Roads + transmission + ports/airports on |
| 7 | Geophysics | Aeromag or 1VD with opacity ~60% over Positron |
| 8 | Everyday tools | Bottom toolbar: Measure line or Annotate pin visible |
| 9 | Pin A Compare | Swipe handle mid-map, badges “A (pinned)” / “Live” |
| 10 | Export / share | Settings → Export or share URL in address bar |

Store finals outside the repo (Drive / Notion) or under `docs/soft-launch/assets/` if you choose to commit compressed JPGs later (keep files small).

**One-liner for LinkedIn / NLPA:**  
Free Labrador critical-minerals map — public layers in one place so you spend time on geology, not fighting the atlas.
