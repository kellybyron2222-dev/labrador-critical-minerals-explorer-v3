# NLPA / prospector one-pager

**Subject:** Free Labrador critical-minerals map (public data in one place)

---

Hi —

Sharing a free, open map built for Labrador prospecting and early siting work.
It pulls the public layers you’d otherwise chase across GeoAtlas / NRCan / CIRNAC
into one browser view — no login, no paywall on the core map.

**Try it:**  
https://kellybyron2222-dev.github.io/labrador-critical-minerals-explorer-v3/

**What you can do in a few minutes**
- Turn on Critical Minerals (MODS) + claims + hard exclusions (parks / protected water) + key infrastructure with **Screen this area**
- Screen a **Radius** around a pin (1 / 5 / 10 / 25 km) or the current map view
- Measure distances, drop pin notes, bookmark views, print, export a ZIP of the current view (WGS 84)
- Compare two setups with **Pin A → Compare** (swipe)

**What it is not**
- Not a Leapfrog / paid discovery platform substitute
- Not Canada-wide yet (Labrador-deep first)
- Not “AI that finds deposits”

**Data:** public government sources (NL GeoAtlas, NRCan, CIRNAC/ISC). About the data is in Settings.
